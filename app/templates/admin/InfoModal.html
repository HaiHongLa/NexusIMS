Home Go to Home

1. Click the icon in the top right corner for help.

2. Map of all available facilities with their names and locations.

Personnel Go to Personnel

1. The Personnel page displays user information.

2. This section allows users to edit their information.

3. Admins can use this section to delete a user.

Products Go to Products

1. This section contains the names of the products.

2. It also shows the availability of products.

3. Users can view detailed information about each product.

4. Users can edit the information of each product.

5. Admins can delete specific products.

Components Go to Components

1. This page allows users to view all components and their properties.

2. Users can view detailed properties of each component.

3. Users can edit the properties of each component.

4. Users can remove components.

5. Users can add more components by clicking "Create" and entering the necessary information.

Facilities Go to Facilities

1. This page allows users to view existing facilities, their locations, and operating status.

2. Users can see whether a facility is operating.

3. Users can view specific details of each facility.

4. Users can modify the information of each property.

5. Users can delete specific properties from the list.

Database Transactions Go to Database Transactions

1. Database Transactions allow users to view all modifications to existing tables, including details about what was changed and when the change occurred.
